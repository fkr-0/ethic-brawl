# diogenes sprite assets

- Put source 4x4 sheet in:         `assets/sprites/diogenes/source/`
- Split into frames:  `./scripts/split_4x4_sprite_sheet.sh diogenes assets/sprites/diogenes/source/<sheet>.png`
- Output frames:  `assets/sprites/diogenes/frames/01.png ... 16.png`
- Motion template:  `assets/sprites/diogenes/meta/motions.template.json`
