# machiavelli sprite assets

- Put source 4x4 sheet in:         `assets/sprites/machiavelli/source/`
- Split into frames:  `./scripts/split_4x4_sprite_sheet.sh machiavelli assets/sprites/machiavelli/source/<sheet>.png`
- Output frames:  `assets/sprites/machiavelli/frames/01.png ... 16.png`
- Motion template:  `assets/sprites/machiavelli/meta/motions.template.json`
