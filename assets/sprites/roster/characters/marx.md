# Karl Marx

## Short Bio
Revolutionary pressure fighter who builds momentum through crowd-control and sustained combos.

## Visual + Animation Guidelines
- Heavy coat, iconic beard, red scarf accent, pamphlet bundle.
- Breathing idle: beard/coat bounce and fist clench cycle.
- Walk: determined marching cadence.
- Run: forceful charge with shoulder-forward posture.
- Run attack should look like barricade-breaking contact.
- Run+jump attack should be an explosive forward plunge.

## Special Attacks (6)
1. Critique Hook (`D + ATK`): close-range body blow chain starter.
2. Das Kapital Wave (`D + F + ATK`): ground-travel shockwave.
3. Class Shield (`D + B + ATK`): temporary guard with retaliatory spark.
4. Manifesto Upper (`JMP + ATK`): rising anti-air punch.
5. Barricade Break (`RUN + ATK`): shoulder ram into hammerfist.
6. Red Comet (`RUN + JMP + ATK`): diving double-fist impact.
