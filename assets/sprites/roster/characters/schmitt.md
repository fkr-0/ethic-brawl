# Carl Schmitt

## Short Bio
Authoritarian control specialist with zoning tools and punish windows around commitment decisions.

## Visual + Animation Guidelines
- Dark formal coat, gloves, severe posture, document-case prop.
- Breathing idle: minimal motion, controlled chest rise, glove tighten.
- Walk: strict heel-first pace.
- Run: short bursts, efficient stride.
- Run attack should be decisive and linear.
- Run+jump attack should be a high-commitment verdict strike.

## Special Attacks (5)
1. Decision Cut (`D + ATK`): quick diagonal slash.
2. Exception Zone (`D + F + ATK`): temporary field that slows enemies.
3. Sovereign Guard (`D + B + ATK`): armored block and riposte.
4. State Advance (`RUN + ATK`): dashing strike that wall-bounces near edge.
5. Verdict Drop (`RUN + JMP + ATK`): airborne axe-kick with hard knockdown.
