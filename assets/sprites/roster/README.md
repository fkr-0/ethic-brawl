# Ethic Brawl Roster Sprite Pack

This folder contains a full roster prompt/spec pack for:
- focault
- aristotle
- socrates
- aquinas (Thomas Aquinas)
- anselm (Anselm of Canterbury)
- marx (Karl Marx)
- bakunin (Mikhail Bakunin)
- schmitt (Carl Schmitt)
- kant (Immanuel Kant)
- hegel

## Files

- `roster.sprite-spec.yml`: master roster sheet, animation blueprint, and command grammar.
- `characters/*.md`: clean character sheets with short bio, animation guidance, and special attack descriptions.

## Animation Standard

Base control inspiration follows a Little Fighter style:
- block / defend
- directional movement
- jump
- attack
- directional combos (`D` + direction + attack)
- run attacks
- run + jump attacks

Each character sheet includes move prompts that can be fed into sprite-generation workflows.
