# leibniz sprite assets

- Put source 4x4 sheet in:         `assets/sprites/leibniz/source/`
- Split into frames:  `./scripts/split_4x4_sprite_sheet.sh leibniz assets/sprites/leibniz/source/<sheet>.png`
- Output frames:  `assets/sprites/leibniz/frames/01.png ... 16.png`
- Motion template:  `assets/sprites/leibniz/meta/motions.template.json`
