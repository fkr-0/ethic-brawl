# camus sprite assets

- Put source 4x4 sheet in:         `assets/sprites/camus/source/`
- Split into frames:  `./scripts/split_4x4_sprite_sheet.sh camus assets/sprites/camus/source/<sheet>.png`
- Output frames:  `assets/sprites/camus/frames/01.png ... 16.png`
- Motion template:  `assets/sprites/camus/meta/motions.template.json`
