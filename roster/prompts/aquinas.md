# Thomas Aquinas

## Short Bio
Scholastic heavyweight combining patient defense with high-impact theological finishers.

## Visual + Animation Guidelines
- Dominican habit, broad frame, chained codex at belt.
- Breathing idle: deep torso rise, cloth weight settling each loop.
- Walk: heavy grounded steps.
- Run: momentum build-up, fewer but longer frames.
- Run attack should feel like unstoppable advance.
- Run+jump attack should have long anticipation and heavy landing.

## Special Attacks (6)
1. Summa Strike (`D + ATK`): armored forearm bash.
2. Five Ways (`D + F + ATK`): five-hit radiant burst cone.
3. Contra Gentiles (`D + B + ATK`): defensive aura reducing ranged damage.
4. Scholastic Hammer (`JMP + ATK`): overhead codex smash.
5. Cathedral Charge (`RUN + ATK`): armored rush with knockback.
6. Angelic Doctor Descent (`RUN + JMP + ATK`): high arc body-drop with quake.
