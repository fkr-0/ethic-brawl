# Socrates

## Short Bio
Athenian questioner whose style baits mistakes and punishes overcommitment.

## Visual + Animation Guidelines
- Barefoot tunic, weathered cloak, expressive beard and hands.
- Breathing idle: chin tilt and open-palmed taunt loop.
- Walk: light circular steps, conversational posture.
- Run: surprising burst with cloak snap.
- Run attack should begin with a feint.
- Run+jump attack should look like a vaulting challenge.

## Special Attacks (5)
1. Elenchus Tap (`D + ATK`): quick interrupting palm.
2. Aporia Pulse (`D + F + ATK`): stun wave with short range.
3. Irony Veil (`D + B + ATK`): evasive backstep counter.
4. Agora Rush (`RUN + ATK`): feint step into elbow strike.
5. Hemlock Plunge (`RUN + JMP + ATK`): descending two-hand slam.
