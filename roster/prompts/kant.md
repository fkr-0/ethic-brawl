# Immanuel Kant

## Short Bio
Precision tactician focused on spacing, strict timing, and principled punish sequences.

## Visual + Animation Guidelines
- 18th-century coat, powdered hair, compact rapier-cane silhouette.
- Breathing idle: formal posture correction loop.
- Walk: neat measured steps.
- Run: controlled quickstep, coat tails follow rhythmically.
- Run attack should look exact and geometric.
- Run+jump attack should be a clean parabolic dive.

## Special Attacks (6)

2. Pure Reason Arc (`D + F + ATK`): crescent slash projectile.
3. Duty Stance (`D + B + ATK`): timed parry with bonus if precise.
4. Noumenon Split (`JMP + ATK`): aerial split-strike.
5. Königsberg Dash (`RUN + ATK`): precise thrust with reposition.
6. Transcendental Drop (`RUN + JMP + ATK`): descending strike that resets spacing.
