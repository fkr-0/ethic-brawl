# Hegel

## Short Bio
Momentum orchestrator who chains stance transitions into escalating synthesis finishers.

## Visual + Animation Guidelines
- Academic coat, broad stance, book-and-cane hybrid weapon.
- Breathing idle: torso sway in three-beat rhythm.
- Walk: deliberate lateral pacing.
- Run: accelerating gait that visually “builds argument”.
- Run attack should start thesis and end antithesis.
- Run+jump attack should culminate in synthesis impact.

## Special Attacks (7)
1. Thesis Tap (`D + ATK`): starter hit, opens combo branch.
2. Antithesis Break (`D + F + ATK`): advancing breaker slash.
3. Dialectic Guard (`D + B + ATK`): absorbs one hit and returns pulse.
4. Logic Spiral (`JMP + ATK`): rotating aerial staff swing.
5. Geist March (`RUN + ATK`): multi-hit advancing sequence.
6. Absolute Dive (`RUN + JMP + ATK`): heavy descending finisher.
7. Synthesis Crown (`D + F + ATK, ATK`): follow-up ender after confirmed hit.
